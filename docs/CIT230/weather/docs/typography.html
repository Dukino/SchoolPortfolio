Typography

Font Choices

Open Sans - This font is used for all headings.

Roboto - This font is used for all non-heading text.

These two fonts have been selected because of their simplicity and how they are easy to read in their respective uses. Open Sans is great for headings because of how it looks when its formated in headings while Roboto is great for paragraphs of text because of how easy it is to read.
