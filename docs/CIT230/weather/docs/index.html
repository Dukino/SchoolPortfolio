Site Name

The website will be called "Rain or Shine" or RainOrShineNow.com. This will be a weather forecast website.

Purpose

The purpose of RainOrShineNow.com will be to provide accurate and timely weather forecasts to users. This is fulfilled through constantly updated weather data and easy to understand layouts and fonts.
